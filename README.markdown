dotvim
======
Contains various . files e.g. `.screenrc`, `.gitignore`

Installation
------------
    ## Clone Repo
    $ mkdir ~/bin
    $ cd ~/bin
    $ git clone https://jldbasa@github.com/jldbasa/dotfiles.git
    
    ## Symlink various file e.g.
    $ ln -s ~/bin/dotfiles/zshrc ~/.zshrc
